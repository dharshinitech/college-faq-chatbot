# College FAQ Chatbot

This project is a College FAQ Chatbot built as part of an internship task.
It helps students by answering common college-related questions using NLP techniques.
## Features
- Answers common college-related questions
- Uses NLP and cosine similarity for matching
- Simple and interactive Streamlit UI
## Technologies Used
- Python
- Streamlit
- NLTK
- Scikit-learn
